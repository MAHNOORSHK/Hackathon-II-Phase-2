# Next.js Skill

This skill provides utilities and guidance for developing applications using Next.js.

## Capabilities

- **Project Setup:** Assist with creating new Next.js projects and configuring basic settings.
- **Component Creation:** Generate Next.js components (pages, API routes, layout, etc.) with best practices.
- **Data Fetching:** Provide guidance and examples for various data fetching strategies (SSR, SSG, CSR).
- **Styling:** Help with integrating and using different styling solutions (Tailwind CSS, CSS Modules, Styled-components).
- **Deployment:** Offer advice on deploying Next.js applications to various platforms.
- **Troubleshooting:** Assist in debugging common Next.js issues.

## Usage

You can invoke this skill to get help with specific Next.js tasks or to generate code snippets.

**Example Prompts:**

- "Create a new Next.js page at `/dashboard`."
- "Show me how to fetch data using `getServerSideProps`."
- "Explain how to set up Tailwind CSS in a Next.js project."
- "What are the best practices for internationalization in Next.js?"
